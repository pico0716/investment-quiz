## 本文ドラフト

投資を始めようとしたとき、最初にぶつかる壁は「何を買えばいいかわからない」じゃないと思っている。

本当の壁は「自分がどんな投資をしたいのかわからない」ことだ。

長期で持ち続けたいのか、短期で回したいのか。
テクノロジー系が好きなのか、安定した配当が欲しいのか。

ここが定まっていないまま銘柄を買うと、値動きのたびに「売るべきか、持つべきか」で迷い続ける。
それは銘柄の問題じゃなくて、スタイルが決まっていないだけだったりする。

---

そう気づいてから、診断ツールを作ることにした。

10問に答えると、投資スタイルを8タイプで診断してくれるやつ。
技術スタックは HTML/CSS/JavaScript のみ。サーバーもDBも不要。
Claude と一緒に設計から実装まで1日かからずに完成させた。

→ 【診断ツールのURL: https://pico0716.github.io/investment-quiz/】

---

## 診断の仕組み

2軸で判定している。

- **軸A（投資目的）**: 長期 or 短期
- **軸B（興味テーマ）**: テクノロジー / 高配当インフラ / グローバル / ESG

10問のうち4問が軸A判定、5問が軸B判定、最後の1問が「証券口座を持っているか」のスクリーニング。
口座の有無で、結果画面のCTAが変わる設計にしている。

---

## Claudeと作ったプロセス（PPCのデモとして）

設計から公開まで、Claude Codeで進めた。

1. 診断ロジックの設計（どう8タイプに分岐させるか）
2. data.js / quiz.js / index.html / style.css の実装
3. GitHub Pages での公開
4. 結果画面へのキャラクター（猫）とSNS誘導の追加

ポイントは「差し替えポイントを data.js に集約したこと」。
テキストや外部リンクを後から変えるとき、quiz.js や HTML を触らなくていい。

こういう設計をClaudeに提案させて、自分で判断して進める。それがPPCの実践例の一つ。
